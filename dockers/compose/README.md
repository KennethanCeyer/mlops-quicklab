<h1 align="center">MLOps Quicklab</h1>
<p align="center">Docker Compose</p>

## Installation

```bash
git clone https://github.com/KennethanCeyer/mlops-quicklab.git
cd dockers/compose/
```

## Getting started

```bash
docker compose up

# detach mode
# docker compose -d
```

## Cleanup

```bash
docker compose down --volume --rmi all
```
